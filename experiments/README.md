# Experiments

These scripts are alternate detector experiments kept for tuning and comparison.

- `golf_ball_detector_gui.py`: larger GUI detector with snapshot annotation.
- `golf_ball_detector_grass.py`: orange golf ball detection tuned for green grass backgrounds.
- `golf_ball_detector_color.py`: color-focused detector experiment.
- `circle_detector_bw.py`: black/white circle detector experiment.

For normal use, start with `../src/orange_detector.py`.
